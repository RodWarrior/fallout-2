    • Instruções de instalação
  • 1) Instalar o jogo Fallout 2 no modo full
  • 2) Antes de instalar a tradução é necessário que instale o Fallout 2 Restoration Project (version 2.3.3). A tradução funcionará corretamente somente após a instalaçao do patch acima.
  • 3) Selecione a pasta raiz de onde você instalou o jogo Fallout 2.
    Exemplo:
    C:\Arquivos de programas\BlackIsle\Fallout2
  • 4) Recomenda-se pular a introdução (pressione ESC três vezes, até aparecer o menu principal), e alterar a seção de legendas no menu de opções para "Sim". Então assista a introdução com as legendas.
